- s[1:4]   == 녕하세

- s[1:-2]  == 녕하

- s[1:]    == 녕하세요

- s[:]     == 안녕하세요 (사본리턴)

- s[1:100] == 녕하세요

- s[-1]    == 요

- s[-4]    == 녕

- s[:-3]   == 안녕

- s[-3:]   == 하세요

- s[::1]   == 안녕하세요

- s[::-1]  == 요세하녕안

- s[::2]   == 안하요
